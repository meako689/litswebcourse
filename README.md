## Django unchained
open terminal 
### Setup
```
git clone https://github.com/meako689/litswebcourse
cd litswebcourse/django-unchained
source setup.sh
```
### populate DB
```
cd newstalk
python manage.py fetchnews
```

### runserver
```
cd newstalk
python manage.py runserver
```

### Run frontend development part
```
cd newsfront
npm start
```
